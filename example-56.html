CSS Color Tutorial

Color Terminology

  • Hue – Refers to color.
  • Saturation – The amount of gray in a color. A maximum saturation there would be no gray in the color, at minimum saturation , the color would be mostly gray.
  • Brightness/Value/Lum – How much black is in a color. At maximum brightness, there would be not black in the color, at minimum brightness it would be mostly black.

What are the ways to refer to color?

  1. A predefined color name
  2. A RGB value
  3. A hex code
